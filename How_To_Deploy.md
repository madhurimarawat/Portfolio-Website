# 🚀 How to Deploy

> [!NOTE]
> Before deploying, make sure you have completed customization.

👉 First complete customization here:  
📄 [How to Customize](./How_To_Customize.md)

---

## 🌐 1. Go to GitHub Repository Settings

- Open your GitHub repository
- Click on **Settings** (top menu)

---

## 📄 2. Open Pages Section

- In the left sidebar, scroll down
- Click on **Pages**

---

## 🔧 3. Configure Deployment Source

- Under **Source**, select:
  - `Deploy from a branch`

---

## 🌿 4. Select Branch & Folder

- Choose branch: `main` (or `master`)
- Select folder:
  - `/root` → if `index.html` is in root
  - `/docs` → if your project is inside docs folder

> [!TIP]
> Always ensure your `index.html` is correctly placed in the selected folder.

---

## 🚀 5. Auto Deployment

- Click **Save**
- GitHub will automatically build and deploy your site

> [!NOTE]
> It may take a few seconds to a few minutes for the site to go live.

---

## 🌍 6. Get Your Live Link

- After deployment, GitHub will show your live URL:

[https://your-username.github.io/repository-name/](https://your-username.github.io/repository-name/)

---

## ⚙️ 7. Enable Website Display (About Section)

- Go back to your repository main page
- Click the **⚙️ gear icon** in the “About” section
- Tick the checkbox:
- ✅ “Use your GitHub Pages website”

---

## 🎉 Done!

> [!NOTE]
> Your portfolio is now live and accessible worldwide 🚀

---

> [!TIP]
